## KEY FEATURES

### Auto Tenant Filtering
All hooks automatically use `coachingId` from TenantProvider:
```typescript
const { coachingId } = useTenant();
const { data: users } = useUsers();  // Automatically filtered by coaching_id
```

### React Query Benefits
- ✅ Automatic caching
- ✅ Background refetching
- ✅ Loading/error states
- ✅ Optimistic updates
- ✅ Query invalidation

### Type Safety
All services and hooks use proper TypeScript types:
- `User` from `@/types/user`
- `Batch` from `@/types/batch`
- `Course` from `@/types/course`
- `Announcement` interface

### RLS Integration
Services use Supabase queries that respect RLS policies:
- Users can only see their coaching's data
- Role-based permissions enforced
- Multi-tenant isolation guaranteed

## USAGE EXAMPLES

### In Components

**Fetch Data:**
```typescript
import { useUsers, useBatches, useCourses } from '@/hooks';

function StudentList() {
  const { data: students, isLoading } = useUsers('student');
  
  if (isLoading) return <div>Loading...</div>;
  
  return students?.map(student => (
    <div key={student.id}>{student.full_name}</div>
  ));
}
```

**Create Data:**
```typescript
import { useCreateBatch } from '@/hooks';

function CreateBatchForm() {
  const createBatch = useCreateBatch();
  
  const handleSubmit = async (formData) => {
    await createBatch.mutateAsync({
      name: formData.name,
      exam_goal: formData.examGoal,
      // coaching_id automatically added
    });
  };
  
  return <form onSubmit={handleSubmit}>...</form>;
}
```

**Filter by Exam Goal:**
```typescript
import { useCoursesByExamGoal } from '@/hooks';

function CourseList({ examGoal }) {
  const { data: courses } = useCoursesByExamGoal(examGoal);
  
  return courses?.map(course => ...);
}
```
